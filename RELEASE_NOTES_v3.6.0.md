# SecuredLanding v3.6.0

## Admin workspace redesign
- Converts major admin areas into expandable accordion folders.
- Converts each loan application into an expandable loan card.
- Adds loan-specific property photo panels inside the matching loan.
- Keeps borrower video review inside the same loan workspace.
- Reduces mobile scrolling and makes queues easier to navigate.

## Database
No new SQL migration is required for this release. It uses the existing `admin_list_property_photos` function and property photo schema from v3.5.1.
